thoughts is a super minimal notes app that i built for myself.

it literally just has a single feature - a text editor in which you write with bullet points. 

it is local first with an optional backend that you self host. i plan to open source this as well.

it also saves some metadata about the note - the date, time, and location.

built in swiftui + uikit for custom animations. the app is super lightweight.

try it out on [testflight](https://testflight.apple.com/join/RnN1rkn3).
